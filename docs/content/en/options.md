### `url`

URL of the Directus server.

```js{}[nuxt.config.js]
export default {
	directus: {
		url: 'https://your-directus-server.app/'
	}
};
```

You can also set Environment variable `DIRECTUS_URL` to override `url`

### `token`

Auth option for static tokens, *user token after login has priority*

```js{}[nuxt.config.js]
export default {
	directus: {
		token: 'sDhjk!.Ddj........'
	}
};
```
